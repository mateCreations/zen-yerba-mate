Dark theme inspired by the yerba mate gourd — earthy greens, warm browns, and steel blues.

|------------|-----------|---------|
| Background | `#1c1e13` | Verde-Oliva Chumbo |
| Surface    | `#282d1c` | Shadow surface |
| Foreground | `#dce0d9` | Cinza Prata |
| Accent     | `#a67c52` | Marrom Ocre / Cuia |
| Border     | `#4f5b4a` | Verde Musgo |

## Installation

Install directly from the [Zen Browser theme marketplace](https://zen-browser.app/themes).

Part of the [Mate theme family](https://github.com/mateCreations) — available for Neovim, VSCode, Obsidian, and Omarchy.
